DTMF decoder

author: Deniz Sincar

This is a simple DTMF decoder that uses WebAssembly to decode the DTMF tones. It uses the dtmf_detector rust crate. The decoder is not perfect, but it works well enough for most cases.

Source code is available on my github

Instructions

  1. allow microphone access
  2. Press a key on your phone or another device. Make sure the microphone on this device hears the dialing tones
  3. See the decoded key below

When you start shouting out the keys that a person nearby is dialing, he will be shocked. This is a fun way to show how DTMF works.
